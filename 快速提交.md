# ⚡ 1分钟快速提交指南

## 最简单的方法 - 直接用浏览器提交！

### 第一步：打开项目
访问：https://github.com/stomfrom/dongzuo

### 第二步：找到要放文件的位置
1. 点击 `动作目录` 文件夹
2. 建议创建一个以你名字命名的文件夹（点击 "Add file" -> "Create new file"，在文件名里写 `你的名字/.gitkeep` 然后提交）

### 第三步：上传文件
1. 点击 "Add file" -> "Upload files"
2. 把你的文件拖拽进去
3. 写个简单的说明，比如 "添加我的动作文件"
4. 点击 "Commit changes"

### 第四步：完成！
就是这么简单！我会看到你的文件并合并到项目中 🎉
